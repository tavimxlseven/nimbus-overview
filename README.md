# ☁️ Nimbus

Nimbus is a cloud automation platform focused on virtual machine provisioning and infrastructure management using Microsoft Azure.

## 🚀 Overview
Nimbus allows users to:
- Deploy virtual machines automatically
- Manage infrastructure efficiently
- Scale cloud environments on demand

## ⚙️ Technologies
- Microsoft Azure
- Virtual Machines (CPU & GPU)
- Infrastructure Automation
- Backend Systems

## 🔒 Private Infrastructure
The core implementation of Nimbus is maintained in private repositories for security and proprietary reasons.

## 📌 Note
This project is actively under development and relies on Azure services for infrastructure provisioning and automation.
